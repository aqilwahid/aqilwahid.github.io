# aqilwahid.github.io

Ini adalah repositori untuk website pribadi saya, **aqilwahid.github.io**, yang dibangun menggunakan **HTML**, **CSS**, dan **JavaScript**, dan di-host menggunakan **GitHub Pages**. Website ini digunakan sebagai portofolio untuk menampilkan proyek-proyek saya, berbagi artikel blog, dan menyediakan informasi kontak.

## Fitur Website

- **Portofolio Proyek**: Halaman ini menampilkan beberapa proyek terbaik yang pernah saya kerjakan, dilengkapi dengan deskripsi, teknologi yang digunakan, dan tautan ke repositori atau demo.
- **Blog**: Saya menulis artikel terkait pengembangan web, teknologi terbaru, pengalaman pribadi, dan lainnya. Blog ini adalah cara saya berbagi pengetahuan dan wawasan.
- **Tentang Saya**: Halaman ini memberikan informasi singkat tentang latar belakang, keahlian, dan minat saya.
- **Kontak**: Informasi kontak dan tautan media sosial untuk memudahkan orang lain terhubung dengan saya.

## Teknologi yang Digunakan

- **HTML5**: Untuk struktur halaman web.
- **CSS3**: Untuk desain dan tata letak halaman web.
- **JavaScript**: Untuk interaktivitas dan fungsionalitas website.
- **GitHub Pages**: Website ini di-host langsung menggunakan GitHub Pages.

## Cara Menjalankan Proyek ini Secara Lokal

Jika Anda ingin mengkloning repositori ini dan menjalankannya secara lokal, ikuti langkah-langkah berikut:

1. Clone repositori ini:

   ```bash
   git clone https://github.com/aqilwahid/aqilwahid.github.io.git
   ```

2. Buka folder proyek:

   ```bash
   cd aqilwahid.github.io
   ```

3. Buka file `index.html` di browser Anda untuk melihat halaman utama website.

## Kontribusi

Jika Anda ingin berkontribusi pada pengembangan website ini, Anda dapat mengikuti langkah-langkah berikut:

1. Fork repositori ini.
2. Buat branch baru untuk perubahan Anda:

   ```bash
   git checkout -b fitur-baru
   ```

3. Lakukan perubahan yang diinginkan dan commit perubahan tersebut:

   ```bash
   git commit -m "Menambahkan fitur baru"
   ```

4. Push branch Anda:

   ```bash
   git push origin fitur-baru
   ```

5. Buat **Pull Request** untuk meminta agar perubahan Anda digabungkan.
